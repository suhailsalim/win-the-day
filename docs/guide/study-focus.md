# Study & focus

## Study timer

Track study/work sessions against a daily hours target:

- Start a session from Today; a **Live Activity** puts the running timer on your Lock Screen and
  Dynamic Island with controls.
- Completed time accrues to the day's study hours, which can auto-satisfy a linked habit and fill
  a custom study ring.
- Manage subjects and review totals in the study management screen.

## Focus screen

A full-screen, distraction-free mode built for deep work (and ADHD-friendly by design):

- A large **depleting ring** shows time remaining — time made visible, ticking entirely on-device.
- **One task at a time**: a single Now/Next display instead of a list, cutting task-switching cost.
- Longer default blocks (45–50 minutes) rather than rapid-fire pomodoros, with gentle,
  non-shaming nudges if you pause.
- Sessions land in your day's focus/study totals.

## Planning tie-in

Routine blocks (see [Planning](planning.md)) can schedule recurring study/work sessions with
reminders, so the timer starts from a plan instead of willpower.
